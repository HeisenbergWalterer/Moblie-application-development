# WeChatCopy - 微信克隆应用

一个使用 SwiftUI 开发的微信克隆应用，采用 MVVM 架构模式。

## 项目架构

- **Models**: 数据模型层（User, Message, Chat, Contact, Moment）
- **Views**: 视图层（界面展示）
- **ViewModels**: 视图模型层（业务逻辑）
- **Services**: 数据服务层（DataService 单例）
- **Utils**: 工具类（扩展、图片选择器等）

## 技术特性

- **框架**: SwiftUI
- **架构**: MVVM
- **响应式编程**: Combine
- **UI 特性**: 
  - 详情页自动隐藏 TabBar
  - 键盘自动管理
  - 自定义颜色主题
  - 流畅的动画效果
